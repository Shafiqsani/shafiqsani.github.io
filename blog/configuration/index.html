Setting up the building environment

  • VS CODE should be install first
  • Download Docker for virtualization of resources
  • Download Qemu for running the operating system
  • WSL 2 follow the guideline for installation
  • Enable the Virtualization from system boot if not enabled
  • Setting up the path for both docker and Qemu in the system variables

How to run the respository

download the respository

you can download the respository from github link

Build an image

Build an image for our operating system using following command

docker build buildenv -t myos-buildenv

COPY the respository

Now copy the downloaded repository to the already created image folder

Build the image

Now run following commands in VS CODE terminal to build the image

docker run --rm -it -v "${pwd}:/root/env" myos-buildenv

make build-x86_64

Now iso image are created in the dist folder after successful execution run exit command

Run the Qemu

Execute the following command to run the OS

qemu-system-x86_64 -L "C:\Program Files\qemu" -cdrom dist/x86_64/kernel.iso
